# 🚀 Manjora AI

Manjora AI is an AI-powered learning and productivity platform designed for students, UPSC aspirants, and professionals.

It helps users with:
- 📚 Study Planning
- 🤖 AI Mentor Chatbot
- 📰 Current Affairs
- 📅 Smart Timetable Generation
- 🔔 Reminder System
- 🏆 Study Streak Tracking
- 📄 PDF Export
- ⏳ Pomodoro Timer
- 📊 Productivity Analytics
- 🎯 Career Roadmaps
- 📘 Book Recommendations

---

# 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Chart.js
- ReportLab

# 🌟 Features

## 🤖 AI Mentor
Interactive AI chatbot for study and career guidance.

## 📅 Smart Timetable
Automatically generates study schedules.

## 📊 Productivity Dashboard
Tracks tasks, hours, and analytics.

## 🔔 Reminder System
Manage study reminders and productivity goals.

## 📰 Current Affairs
Displays important education and technology news.

## 📄 PDF Export
Download study reports as PDF.

## 🌙 Dark Mode
Modern responsive UI with dark mode support.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/manjoraai-hub/ManjoraAI.git
```

## Open Project

```bash
cd ManjoraAI
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
